fmt-cmd

(fmt-cmd paths {:keys [check write canonical edn]})

Format command: verify roundtrip fidelity or reformat. With –check: verify file(s) roundtrip without modification. With –write: format file(s) in place. With –canonical: format to canonical style and print to stdout. With –edn: output results in EDN format. Returns exit code 0 for success, 1 for failure, 2 for no files/path error.
