shiftlefter.gherkin.diagnostics

Centralized error formatting for consistent diagnostic output.

Standard format: path:line:col: :

Usage: (format-error path error) ; single error string (format-errors-human path errors) ; multi-line human output (format-result-edn result) ; machine-readable EDN

Count errors by type. Returns map of type -> count.

format-error

(format-error path {:keys [type message location], :as _error})

Format a single error in standard diagnostic format. Format: path:line:col: :

If path is nil, uses ‘-’ (stdin convention). If location is missing, uses line 1, col 1.
